# Gesture Detector

MediaPipe-based gesture detection with custom meme overlays.

## Live Demo

Visit: **https://aaronhubhachen.github.io/gesture-detector/**

## How It Works

- Shows a blank monkey when no hand is detected
- Shows a gun gesture monkey when a hand is detected
- Shows a thinking monkey when your finger is near your mouth

## Setup

To enable GitHub Pages:
1. Go to repository Settings
2. Navigate to Pages section
3. Select "Deploy from main branch"
4. Save and wait 1-2 minutes for deployment

## Files

- `index.html` - Main application
- `blank.jpg` - Default state
- `smile.jpg` - Hand detected
- `think.jpg` - Finger near mouth
